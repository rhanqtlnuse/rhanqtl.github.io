排序是编程中经常出现的场景，C++ STL 为我们提供了一些用于排序的函数。

sort 函数

STL 中最常用的排序函数是 algorithm 中的 sort，它的声明如下：

sort 函数将 [first, last) 之间的元素按照升序进行排列。其中，第一个重载使用重载的 < （operator<）进行比较，第二个重载使用 comp 进行比较，comp 可以是一个形如

注意点

  • sort 函数是不稳定的排序，可以使用 stable_sort 进行稳定排序
    • stable_sort 的原型与 sort 相同
    • sort 的复杂度是 $n\log(n)​$，stable_sort 的复杂度为取决于具体情况，如果有足够的额外空间，算法的复杂度为 $n\log(n)​$；否则，算法的复杂度为 $n\log^2(n)​$
  • sort 可以用于如下的容器：
    • array
    • vector
    • deque
    • 关键是 RandomAccessIterator
  • sort 同样也可以用于 C-style 数组
  • 有时想要将容器中的元素逆序排列，一种方法是先 sort 然后 reverse，但是还有其他的办法：
    • 使用容器的 rbegin() 和 rend()，逆向迭代容器中的元素
    • 修改 comp 参数

is_sorted 和 is_sorted_until

这两个函数是 C++ 11 中新增的。

Rearranges the elements in the range [first,last), in such a way that the element at the nth position is the element that would be in that position in a sorted sequence.

The other elements are left without any specific order, except that none of the elements preceding nth are greater than it, and none of the elements following it are less.
