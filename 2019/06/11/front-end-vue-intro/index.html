组件 CSS 作用域

在组件的 <style> 中设置的样式会被应用于全局，使用 scoped 将样式的作用域限制为当前组件。

具体实现：给 style 设置一个标识符，然后在对应的标签中引用。比如：

{ asset_img “scoped-style.png” }
