磁盘

什么是磁盘？

图中的“Track”就是磁道，磁道是由一个个的扇区（sector，512 字节大小的块）组成的。

为了提高效率，许多文件系统一次读/写 4KB（或者更多），但是只能保证一个扇区的写是原子性的。

只要知道（盘面，磁道，扇区）就可以定位磁盘上的一个存储块。
