C/C++ 中的计时函数

我们可能经常遇到下面这样的需求：写了一个算法，需要计算运行时间；获取一个时间戳等等。在 Java 中，我们可以使用 System.currentTimeMillis() 和 System.nano()，那么在 C 和 C++ 中如何实现呢？本文介绍三种用于计时的函数。
