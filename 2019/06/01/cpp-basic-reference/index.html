输出为：

    sizeof(a) = 4, address of a = 0x61ff18
    sizeof(ra) = 4, address of ra = 0x61ff18

那么问题来了：引用在底层到底是什么？

我们来看汇编：

-16(%ebp) 就是我们的局部变量 a，-12(%ebp) 就是引用变量 ra，可以看到，ra 中保存的实际上是 a 的地址，所以引用在底层实际上就是一个指针。

这样一来，我们可以借助指针的概念来理解引用。

int &ra = a; 实际上类似于 int * const p = &a;
