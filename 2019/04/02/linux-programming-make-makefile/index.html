自动变量

变量 含义
$@ 目标的完整名称
$< 第一个依赖文件的名称
$? 所有比目标新的依赖文件名，以空格隔开
若依赖文件是档案文件的成员，只使用成员的名称
$^ 所有依赖文件名，无重复，以空格隔开
若依赖文件是档案文件的成员，只使用成员的名称
$+ 与 $^ 相似，但不会去重
对于链接命令中以特定顺序重复出现的库文件名有作用
$*
$% 如果目标是归档成员，则该变量表示目标的归档成员名称

记忆方法：

  • $?：可以记忆为向 make 询问，哪些依赖文件会被重新编译 ?
  • $^：在程序设计语言中，^ 通常表示抑或。可以将这个变量记忆为，如果出现重复（假设依赖名称为 “a”），那么 a ^ a = 0，得到空，不会将非首次出现的名称放入列表中
  • $+：可以记忆为列表的追加
