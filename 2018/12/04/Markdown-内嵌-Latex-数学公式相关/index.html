Markdown 内嵌 Latex 数学公式相关

发表于 2018-12-04 | 浏览 次

Markdown 中通常使用 （行内公式）或 $$$…$$$（行间公式）或 \[...\] 来标明 Latex 的使用

符号

上标：^{}，当为单个符号时可省略 {}

下标：_{}，当为单个符号时可省略 {}

求和：\sum

  • 默认情况下是上标/下标的形式，可以使用 \limits 来变成上/下
    • 默认情况：\sum_{i=0}^n i：$\sum_{i=0}^n i$
    • 使用 \limits：\sum \limits_{i=0}^n i：$\sum \limits_{i=0}^n i$

点

  • \cdot：$\cdot$
  • \cdots：$\cdots$
  • \vdots：$\vdots$

示例：：$\begin{matrix} 1 & 2 \ 3 & 4 \end{matrix}$
