NP 完备性

NP-完备问题是 NP 问题中最难的问题

如果一个问题 P 在集合 NP 中，并且如果存在一个 P 的算法，使得 NP 中的所有问题都能够在有限的额外工作的情况下解决，那么决策问题 P 就是 NP-完备的

说人话：NP-完备问题就是说，某个 NP 问题 $P$ 有一个算法，对于 NP 问题集合中的其他问题，我们能够通过在这个算法中增加一些有限的额外工作令其得到解决，那么这个问题 P 就叫做 NP-完备问题。

也就是说，如果我们对于问题 P 有一个算法，并且我们知道如何将问题 Q 的实例转化为问题 P 的实例，我们也就得到了解决问题 Q 的算法！

多项式规约

如果如下条件：

  • T 可以在多项式时间内计算完成（取决于输入的长度）
  • 输入 x 是 Q 的 “yes” 输入当且仅当 T(x) 是 P 的 “yes” 输入

成立，那么T 是一个从 Q 到 P 的多项式规约

Q 是多项式可规约到 P 的：$Q \le_P P$

P 至少与 Q 一样难

NP-完备问题的另一种定义：如果一个问题 P 在集合 NP 中，并且对于 NP 集合中的每一个问题 Q，他是多项式可规约到 P 的，那么决策问题 P 就是 NP-完备的

NP-完备和 NP 难

如果对于一个决策问题 P 而言，NP 中的每一个问题 Q 都能够多项式规约为 P，那么 P 就是 NP 难的

NP-难问题是至少与 NP 中最难的问题一样难的问题。一个决策问题 P 是 NP-完备的，如果它既是 NP 的又是 NP-难的

一旦我们找到一个第一个 NP-完备的问题，那么找到其他 NP-完备问题就容易得多了：

  1. 首先找到一个 NP-完备问题 P
  2. NP 中所有的问题 Q 都可以多项式规约为 P
